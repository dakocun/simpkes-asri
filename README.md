# simpkes-asri

**Sistem Informasi Manajemen Pelayanan Kesehatan**
*Ners Asri Cantik*

Aplikasi PWA untuk manajemen pelayanan kesehatan perawat desa, mencakup:
- Manajemen stok obat
- Layanan pasien (Kunjungan Rumah, Posyandu, Kantor Desa)
- Laporan RJTP (Rawat Jalan Tingkat Pertama)
- Jadwal shift
- Home care
